# 🍧 Açaí Hut — Website Institucional para Açaiteria

Este projeto foi desenvolvido como um trabalho freelancer para a **Açaí Hut**, com o objetivo de criar uma presença digital moderna, responsiva e atrativa para o negócio.

O foco foi desenvolver um site visualmente impactante, com navegação simples e otimizada para dispositivos móveis.

---

## 📌 Objetivo

> Criar um website profissional para uma açaiteria, com foco em conversão, apresentação dos produtos e experiência do usuário em dispositivos mobile.

O projeto teve como objetivos principais:

* Apresentar os produtos de forma visual e atrativa
* Facilitar o contato e localização do cliente
* Criar uma experiência fluida em dispositivos móveis
* Fortalecer a presença digital da marca

---

## 🧠 O que foi aplicado neste projeto

* Desenvolvimento com abordagem **Mobile First**
* Estruturação de layout focada em **UX/UI**
* Organização de seções para **melhor conversão**
* Uso de **HTML semântico**
* Estilização moderna com CSS
* Interatividade com **JavaScript**
* Boas práticas de responsividade

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** – estrutura da página com marcação semântica
* **CSS3** – estilização com foco em layout, cores e responsividade
* **JavaScript** – interações e динамика da interface
* **Mobile First** – desenvolvimento pensado inicialmente para dispositivos móveis

---

## 📱 Responsividade

O projeto foi desenvolvido utilizando o conceito de **Mobile First**, garantindo uma experiência otimizada para:

* 📱 Smartphones
* 📲 Tablets
* 💻 Desktops

## 🖥️ Como rodar o projeto localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/wemersonmatheuss/acai-hut.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd acai-hut
   ```

3. Abra o arquivo index.html no navegador
   3.1 Clique duas vezes no arquivo `index.html`, ou utilize a extensão *Live Server* no VS Code
